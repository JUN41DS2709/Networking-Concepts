# MAC ADDRESS

* MAC stands for **Media Access Control**.
* MAC address is a type of address which is provided by the manufacturer and comes implemented with the network card of the device.
* It is also called the **physical address** or **hardware address** of the device.
* It is a **12-character** address.
* It is a fixed address which is written in **hexadecimal format** such as `1F:2A:3D:4B:5C:6E`.
* Its main purpose is to route the data inside the local network.
* It works on the **second layer (Data Link Layer)** of the OSI model.

## Example

* Someone from another local network sends data and it reaches the router. Now the router will make use of the MAC address to route and send the data to the receiver device inside its own network.

# IP ADDRESS

* IP stands for **Internet Protocol**.
* IP address is a **32-bit address** which is written in numeric format (**IPv4**).
* IP address is an address which is used to uniquely identify a device in a network.
* IP address is like the postal service of the internet which routes the data through the most suitable path over the network.
* An IP address is made up of **4 sets of numbers**, and each set is of **8 bits**. These sets are called **octets**.
* Unlike a MAC address, an IP address is not a fixed address.
* IP address has three different types: **Private, Public, and Loopback**.
* IP address works on the **Network Layer** of the OSI model.

## Example

* Someone wants to send data outside the network, so the data is assigned a packet which contains the IP address of the receiver and sender. This helps the data travel over the network and reach the destination.
